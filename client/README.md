## Setup
#### Install dependencies
Should be yarn version 3.2.0
```
$ yarn install
```

#### Running local dev server
```
$ yarn dev
```

### Build Setup

```
$ yarn build
```

Build should be outputted to `dist/` folder

After that we can check production version with `yarn preview`

### Main Tech stack

* React
* React-router
* React-Context
* Vite.js
* Typescript
* Ant Design
